# تطبيق الاستعداد للامتحانات – Iraqi Exam Prep

تطبيق احترافي مبني بإطار Flutter لمساعدة طلبة الماجستير في العراق على التحضير للامتحان الوطني من خلال تجربة عربية كاملة وواجهات عصرية.

## المزايا الرئيسة

- **مستخدمون وحسابات**: تسجيل دخول وتسجيل جديد بواجهة عربية، حماية JWT، إدارة البيانات الشخصية.
- **نظام الامتحانات**: أسئلة عشوائية، واجهة تفاعلية، تتبع فوري للتقدم، صفحة نتائج مفصلة ورسائل تشجيعية.
- **الاشتراك المميز**: تدفق شراء عبر تيليغرام، رموز تفعيل، الوصول إلى جميع المواد والمزايا الإضافية.
- **إدارة الأسئلة للمشرف**: لوحة خاصة بالمشرف لإضافة الأسئلة بالعربية مع خيارات متعددة وتحديد الإجابة الصحيحة.
- **صفحة التسجيل الوطني**: عرض خطوات ومتطلبات التسجيل الرسمي وروابط التواصل.
- **تجربة مستخدم عصرية**: تصميم نظيف، ألوان خاصة لكل مادة، دعم كامل لاتجاه النص من اليمين إلى اليسار.

## المتطلبات

- Flutter SDK ‎3.0‎ أو أحدث
- Dart SDK ‎3.0‎ أو أحدث
- Android Studio / Xcode لتشغيل المحاكيات
- خادم الـBackend قيد التشغيل (راجع README الخاص بالنسخة الخلفية)

## البدء السريع

```bash
cd iraqi_exam_prep_app
flutter pub get
flutter pub run build_runner build --delete-conflicting-outputs
```

### إعداد عنوان الـAPI

حرر الملف `lib/core/constants/app_constants.dart` وعدّل ثابت `baseUrl` ليتطابق مع عنوان الخادم لديك (مثلاً `http://192.168.0.243:3000/api/v1`). للمحاكي استخدم `http://10.0.2.2:3000/api/v1`.

### تشغيل التطبيق

```bash
flutter devices        # عرض الأجهزة المتصلة
flutter run -d <id>    # تشغيل التطبيق على جهاز محدد
```

## هيكلية المشروع

```
lib/
 ├─ core/
 │   ├─ constants/      # AppConstants وغيرها من الثوابت
 │   ├─ theme/          # AppTheme و AppColors
 │   ├─ network/        # DioClient وإعدادات الاتصال
 │   └─ di/             # مستودع الاعتماديات (GetIt)
 ├─ features/
 │   ├─ auth/           # المصادقة (شاشات، BLoC، بيانات)
 │   ├─ exams/          # الامتحانات والنتائج
 │   ├─ activation/     # الاشتراك والتفعيل
 │   ├─ registration/   # التسجيل للامتحان الوطني
 │   └─ admin/          # لوحة إدارة الأسئلة
 └─ config/
     └─ routes/         # GoRouter
```
